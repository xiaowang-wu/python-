随机模块

```python
import numpy as np
np.random.rand(3,2)
#array([[0.59916264, 0.38927327],
#      [0.45786288, 0.66356425],
#      [0.45444674, 0.74427271]])
np.random.randint(10,size=(5,4))  // 随机生矩5*4的矩阵，距阵值是[0-10) 
# array([[1, 1, 8, 0],
#       [3, 0, 3, 8],
#       [3, 2, 9, 4],
#       [7, 9, 8, 3],
#       [7, 0, 5, 0]])     
                                                     np.random.randint(0,5,5)  //生成5个0-5的数组    
# array([2, 0, 4, 2, 1])  
z,x=0,0.1
n=np.random.normal(z,x,10)  //高斯分布
n
#array([-0.01037599, -0.07086384,  0.13879497,  0.01044236,  0.15074967,
#        0.06877362,  0.08050338, -0.0160742 ,  0.15557183,  0.09318226])  
np.set_printoptions(precision=3) // 执行之后，后面程序生成的数的精度为 3
                                                     
```

随机种子

随机种子可以用来保存当前产生的随机数

即： 当执行完随机种子再执行随机数产生，此时会保存在当前随机种子种，当下一次执行刚才保存过数据的随机种子，再执行随机数，会得到到随机种子所保存的数据。效果

```python
np.random.seed(0)
z,x=0,0.1
n=np.random.normal(z,x,10)  // 高斯分布
n
# array([ 0.176,  0.04 ,  0.098,  0.224,  0.187,-0.098,  0.095, -0.015,-0.01 ,  0.041])
n=np.random.normal(z,x,10)
n
# array([ 0.014,  0.145,  0.076,  0.012,  0.044, 0.033,  0.149, -0.021,0.031, -0.085])
np.random.seed(0)
z,x=0,0.1
n=np.random.normal(z,x,10)
n
# array([ 0.176,  0.04 ,  0.098,  0.224,  0.187,-0.098,  0.095, -0.015,-0.01 ,  0.041])
np.random.seed(10)  // 这里买10 可以是任何数
np.random.randint(0,8,3)
#array([1, 5, 4])
np.random.randint(0,8,3)
#array([7, 0, 1])
np.random.seed(10)
np.random.randint(0,8,3)
#array([1, 5, 4])
```

此放方法适用于所有产生的随机数不仅仅局限于高斯分布

注：此处可以理解为每一个种子可以保存一组随机数，当需要此随机数时可以调出来进行使用
